# Tier 2 — Views Architecture

Navigator OS evolves from:
Tier 1 → Process Visibility
Tier 2 → Operational Views

## Views

### 1. CEO View
- Company health
- Revenue flow
- Alerts
- Risk detection

### 2. Area View
- Team workload
- Process status
- Bottlenecks
- Performance

### 3. User View
- Tasks
- Checklist
- Approvals
- Deadlines

## Principle

CEO → Sees everything  
Area → Manages execution  
User → Executes tasks
